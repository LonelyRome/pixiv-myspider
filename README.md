前序

-----------

一个基于python的P站爬虫

主要是爬搜索页下的全部图片，还可以根据赞过滤一些作品，但是只是粗略过滤，如果有更好的算法可以进行讨论。

该爬虫不能多线程，虽然我在里面还写有多线程的代码，但我不建议你打开它。毕竟这时候了还想自行车？（滑稽）

当然后续会多一些操作，比如关于动态图的下载。

也欢迎大佬指点qwq
